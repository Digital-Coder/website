# Cloud Overview

Want the power of Umami Analytics without the hassle of managing your own servers?

Umami Cloud uses its own open source analytics platform. Major features will always be developed in parity between Umami Cloud and the open source project.

## Signup

You can signup at [Umami Cloud](https://cloud.umami.is/signup).

New users will automaically be included in our hobby plan, which is a free tier with limited usage.

Details on each plan can be found on the [Pricing](https://umami.is/pricing) page.

## Usage

`Hobby` plan usage is calculated at the beginning and end of each month, regardless of sign-up date.

`Basic` and `Business` subscription usage is calculated for the subscription period.
