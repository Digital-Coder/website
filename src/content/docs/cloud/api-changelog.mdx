# API Changelog

The latest Umami API version is: [0.64.0](https://www.npmjs.com/package/@umami/api-client)

This page documents changes made to the Umami API client.

### 2024-02-16

- `GET /reports/index`: Added websiteId and teamId added as query parameters
- `/teams/[id]/websites/[websiteId]`: DEPRECATED
- `GET /teams/[id]/users/[userId]/index`: Added new GET route
- `POST /websites`: Added teamId as an optional parameter
